# servoMotorDemo

# やりたいこと

- 光センサに連動して、サーボモータを動作させる。
- 明るければ、右に。暗ければ、左に。
- サーボモータの傾きをLCDに表示する。

# 仕様

## サーボモータ (SG90 サーボモータ)
・PWMサイクル：20mS
・制御パルス：0.5ms~2.4ms
・制御角：±約90°(180°)
・配線：茶=GND、赤=電源[+]、橙=制御信号 [JRタイプ]
・トルク：1.8kgf・cm
・動作速度：0.1秒/60度
・動作電圧：4.8V(~5V)
・温度範囲：0℃~55℃
